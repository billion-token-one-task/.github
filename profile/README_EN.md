# Joule — Token-Scale Cognitive Architecture Research

> Token is to cognition what the joule is to energy.
> Each order of magnitude in token orchestration unlocks a new class of previously impossible tasks — the bottleneck is not the model, but the architecture.

> **Our thesis: The bottleneck of long-horizon execution is not single-step intelligence — it is how errors accumulate in state, are detected, and are rolled back.**

---

## What We Do

We are launching a **"Kardashev Project" for AI** — researching how to effectively constrain 10⁹–10¹² tokens toward a **single complex objective**, building next-generation cognitive architectures for AI agents.

Most AI agent systems today operate at 10⁴–10⁶ tokens per task. We work on the frontier: 10⁸ and beyond, where model capability is no longer the bottleneck — **the architecture that orchestrates massive token flows in service of a single task is**.

We don't run 1,000 agents on simple parallel tasks. We orchestrate an agent system around **one extraordinarily complex goal**, executing over weeks or months with minimal or no human intervention, rollback capability, and zero drift.

### Proven Results

- **$36** to complete 9 engineer-months of autonomous code repair
- **$30** to complete 2 researcher-years of cross-disciplinary literature discovery

### Active Research Areas

Three angles into this thesis:

- **Token-scale orchestration** — verification-repair loops, state compression, and efficiency measurement for billion-token tasks
- **Latent-space communication** — replacing MB-level text exchange between agents with KB/byte-level continuous representations (our 512-byte compressor matches full-KV baselines on GSM8K)
- **Agent efficiency science** — empirically mapping the bandwidth-accuracy curve, cognitive Carnot efficiency, and ε (state compression loss) across task families

**Next 4–6 weeks**: Empirically map long-horizon reliability curves and isolate the marginal contribution of error accumulation, detection, and rollback — drawing from our 1,100+ agent trajectories / 32 cross-repo PRs / token-budget probing data.
